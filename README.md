# InAmigos Foundation — NGO Awareness Website

A modern and responsive NGO awareness website developed for the InAmigos Foundation internship task using HTML and CSS.  
This project highlights the foundation’s mission, initiatives, social impact, events, and opportunities for people to volunteer and contribute towards meaningful causes.

---

## 🌍 About The Project

This website was created to spread awareness about the social initiatives and community-driven projects of **InAmigos Foundation**.

The platform focuses on:
- Social impact awareness
- Community engagement
- Volunteer participation
- Women empowerment
- Education support
- Environmental sustainability
- Animal welfare

The design follows a clean, modern, and professional NGO-style interface with responsive layouts and visually engaging sections.

---

## ✨ Features

- Fully responsive modern UI
- Hero section with impact visuals
- NGO introduction and mission
- Causes / Initiatives section
- Community impact statistics
- Events showcase section
- Quotes & social message section
- Contact / Join Us form
- Smooth scrolling navigation
- Mobile-friendly layout
- Organized HTML & CSS structure

---

## 🛠️ Built With

- HTML5
- CSS3
